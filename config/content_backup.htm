利用 cmsimde 建立靜態網誌方法

1. 在 github 建立倉儲, git clone 到近端

2. 參考 https://github.com/mdecourse/newcms, 加入除了 cmsimde 目錄外的所有內容

以 git submodule add https://github.com/mdecourse/cmsimde cmsimde

建立 cmsimde 目錄, 並從 github 取下子模組內容.

3.在近端維護時, 更換目錄到倉儲中的 cmsimde, 以 python wsgi.py 啟動近端網際伺服器.

動態內容編輯完成後, 以 generate_pages 轉為靜態內容, 以 git add commit 及 push 將內容推到遠端.

4. 之後若要以 git clone 取下包含 submodule 的所有內容, 執行:

git clone --recurse-submodules https://github.com/mdecourse/newcms.git

Analysis Required

在electronic packaging (電子設備外殼)的設計中，有許多設計需要進行最嚴格的分析，而在任何競爭激烈的產品設計領域裡，將分析與技術結合將提高公司的生產力，進而大大提高成功的機會。

如果產品需要最嚴格的分析，那他會擁有以下的特徵：

  • 高產量，如果生產成千上萬的個別零件，那節省一元也會帶來可觀的成本節省，儘管分析只對單個零件帶來小部分的成本節省，但由於大量生產的關係，反而會帶來大量的利潤，反之，如果是小量生產，就算是實用性的零件，無論大量投資多少在降低成本上，也不會帶來可觀的利潤。
  • 高安全性，不管是在運輸、公用、醫療或教育行業中，客戶都需要能夠安全操作的產品。
  • 對客戶需求的”功能”，非指一般的零件，而是指包括軍事、航天局或政府所需的高規格產品。

請注意，過度的設計導致降低公司盈利能力的情形是不該出現的，設計師和工程師應該時刻警惕降低成本的可能性。淘汰的零件、生產的能力和整體高貴的設計決定了產品領導力(花費在決定要生產哪些產品以增加對顧客的價值)，所以設計的第一階段是能夠最大程度降低成本的，而隨著設計甚至進入原型階段，為降低成本而重新設計的成本開始呈指數級增長。關於這一方面的內容將在第6章“組裝和可維修性”中介紹。





4.3 內容

材料與流程(加工)的選擇

     一旦設計者設計完零件後，設計者必須找出生產零件的”最佳”成本，

      要找出的最佳成本所需要考慮的是:

• 零件材料的選用

• 處理完成的零件

• 零件所需的尺度精度

• 生產該零件的流程 

    一種是早期所需的流程、原型設計和零件的pre-production(前期生產作業)，

     另一種是對於已經成熟(設計好、經過改良)的零件所進行的流程

• 零件的生產數量

    例如，每個季度，每個月，每年，需要多少的生產數量

• 零件所需的第二次作業(第二次加工)

• 零件的花費成本

• 該零件可以與設計中的另一個零件結合(組合)嗎？

    實質上，需要確定的是單個組合用的零件是否能夠滿足在零件沒有組合時的功能性。

• 零件能夠做成對稱性的嗎(便於組裝)？

  或者把原本幾乎對稱的部分做成明顯非對稱的? 

     這兩個問題涉及零件的組裝以及組裝的可能性，如果只是為了讓零件能夠對稱，

     而在零件上添加 多餘的孔或缺口，那將是一個錯誤的方法

決定選定零件的材料時，所需注意的事項：
設計者應該選擇滿足（達到或超過）以下要求的材料：

  1. 強度
  2. 重量
  3. 可靠性 (P.S.: 產品在規定的條件下和規定的時間內，完成規定任務的概率)
  4. 法規要求 (P.S.:每個監管機構針對產品所有的批准，許可，註冊)
  5. 安全性
  6. 散熱性
  7. 屏蔽能力 (P.S.:抵擋輻射、電磁干擾或射頻干擾的能力)
  8. 金屬的相容性 (P.S.:生物相容性: 材料與生物體之間相互作用)或互適性 (P.S.:與其他材料產生交互作用)
  9. 導電性或絕緣性
  10. 透明性
  11. 磨耗性
  12. 美觀性(觸覺上或視覺上)
  13. 聲音性(P.S.:透音或無透音)
  14. 紫外線（UV）傳輸和電阻要求

簡單亂數抽取程式

首先我們先做一個簡易的亂數抽取程式，主要是利用random模組來進行

random.randrange(100,200,10) 在100~200之間抽取一個10倍數的數字

所以只會抽中100,110,120,130,140.....等數字。

#導入模組
import random
#設定第一個數字
x=input("first number")
#設定第二個數字
y=input("second number")
在第一個數字和第二個數字之間亂數抽取一個數字
print(random.randrange(int(x),int(y),1))

然而這只是一個最基礎的程式，亂數分組程式必須同時包含

1.一次亂數列出許多人，再依照人數來分組。
